.Net程序员开发效率提升大合集

一刀
2022-05-08
.Net, 工具箱

简介

本文来自于我正在GitHub上发布的.Net开发者工具箱项目，项目地址：https://github.com/laosanyuan/dotnet-developer-toolbox ，后期会继续更新更多的优质内容，如有需要可关注原项目。

本项目整理了一些在.Net开发者可能会经常使用到的一些辅助工具，旨在帮助.Net开发者更专注的编码、更高效的工作。

整理的工具和软件包含了开发、学习、求职、生活等各方面，其中绝大多数都为免费或提供免费使用方式的在线或本地工具以及站点。首次上传，可能有些分类尚不合理或内容有所欠缺，后续会不断补充调整，逐渐完善。也欢迎各位好汉提交PR，为中国.Net开发者提供更多的帮助。

.Net相关

  • MSDN - 微软相关的官方技术集中地，主要是文档类
    • .Net文档
    • C#文档
    • 微软C#编码规范

IDE

  • Visual Studio - 宇宙第一IDE，不用解释
  • VS Code - 是一款由微软开发且跨平台的免费源代码编辑器，插件丰富， 最受开发者欢迎的开发环境

调试工具

  • Snoop - 开源的WPF UI探索工具，可以在调试过程中实时浏览WPF程序的属性、Triggers，设置断点等
  • Fiddler - Http代理抓包，本地客户端工具
  • Wireshark - 网络数据包分析软件
  • Bus Hound - USB串口抓包软件
  • FileZilla - FTP上传工具，开源免费
  • MobaXterm - SSH客户端工具，免费。 全能型终端神器，比传统XShell好用
  • Sokit - 数据包收发测试工具，操作简单，开源免费。可模拟TCP/UDP客户端服务端
  • SharpLab.io - 在线获取C#编译中间过程和结果，可深入理解代码，非常好用
  • SoJson - 在线Json解析转换
  • Json.cn - 在线Json解析转换
  • Regexr - 正则表达式验证匹配
  • Ctool - 字符串加解密
  • Yaml在线编辑器 - Yaml在线编辑、格式化
  • 北京时间 - 在线时间戳转换工具
  • 菜鸟工具 - 常用单位换算

构造工具

  • NSIS - Windows平台下可执行文件打包工具
  • ILSpy - 开源的.Net反编译工具
  • dnSpy - 开源的.Net反编译工具
  • dotPeek - JetBrains出品的免费.Net反编译工具
  • .NET Reactor - .Net程序加壳混淆工具，收费软件，提供试用
  • Dependency Walker - dll动态链接库反编译工具
  • dotTrace - .Net性能分析工具

代码托管平台

  • GitHub - 最流行的在线代码托管平台，汇聚了世界范围内非常多的开发者和开源项目
  • Coding - 提供永久免费的基础服务
  • 码云 - 免费提供团队协作、源码托管、代码质量分析、代码评审、测试、代码演示等功能

GitHub辅助工具

  • Shields.IO - GitHub徽章生成
  • .gitignore.io - 为你的项目创建必要的.gitignore文件
  • Gitstar Ranking - GitHub开源项目全球排名
  • GitHub emoji表情列表 - GitHub支持使用的emoji合集

常用UI资源

字体

  • 字体家 - 免费、付费字体授权下载
  • 优优网 - 免费商用字体
  • 自由字体 - 汇聚全网免费字体
  • DaFont - 提供免费英文字体的全球知名站点
  • 1001 Free Fonts - 致力于分享免费英文字体的网站
  • Use & Modify - 各种稀奇古怪英文字体
  • 识字体 - 字体识别
  • VAU Fonts - 在线生成艺术字体，只支持英文内容

配色

  • Picular - 色彩搜索引擎
  • Coolors - 在线快速生成配色

图标/矢量图

  • Font Awesome - 在线获取可缩放矢量图标
  • Blobmaker - 在线SVG形状生成器
  • 阿里巴巴矢量图标库 - 大量免费矢量图标
  • Inkscape - 比较流行的本地矢量图形便捷工具，可以完成各种复杂的矢量图绘制

图库

  • FreeImages - 免费照片与插图下载，所有照片均可免费用于个人和商业用途
  • Pexels - 40000+免费商用图片
  • StockSnap - 海量商用图片免费使用

文档编写

在线笔记

  • 印象笔记 - 支持多端同步、网页剪藏、OCR，构建私人笔记系统
  • 有道云笔记 - 网易旗下专注办公提效的笔记软件，用户可以随时随地对线上资料进行编辑、分享以及协同
  • 语雀 - 阿里出品知识库产品呢， 优雅高效的在线文档编辑与协同工具
  • 石墨文档 - 免费的在线实时协作文档服务，界面简单清晰，多人协同编辑，可以讨论和查看变更
  • GitBook - 是一款现代化的文档平台,支持团队协作,可以在上面写产品文档、内部知识分享、接口文档、电子书等

编辑器

  • Typora - 简单强大的本地Markdown编辑器，所见即所得，跨平台、免费
  • NotePad++ - 比windows自带的记事本更强大，除了适合作为一款轻量型的文本编辑软件，也很适合作为编程使用

绘图

  • ProcessOn - 在线流程图思维导图绘制，有限免费
  • draw.io - 开源在线流程图绘制工具
  • excalidraw - 轻量的在线白板工具，绘制手绘风格图形
  • XMind - 全功能的思维导图和头脑风暴软件，美观易用

工作效率提升

  • 滴答清单 - 清单工具，规划记录工作事项，梳理工作内容

刷题

  • LeetCode - 最常用的刷题网站
  • LintCode - 数量超过 1000 道的算法题目和人工智能题目
  • 牛客网 - 刷题、求职、面经、部分公司机试

课程/教程

  • YAML入门教程
  • JSON教程
  • Markdown教程
  • Git教程
  • 正则表达式教程
  • SVG教程

简历

  • 五百丁 - 汇聚大量简历模板，支持自动生成简历
  • 超级简历 - 可将简历自动导入、排版、纠错、并翻译为多种语言
  • 简历本 - 一个专门写简历的网站

求职

  • Boss直聘 - 互联网行业最常用的招聘/求职平台
  • 脉脉 - 实名制商业社交平台，了解业内新鲜资讯
  • Linkedin - 全世界最大的职业社交网站之一，也是一家面向商业客户的社交网络

社交网站、博客

国内

  • CSDN - 专业开发者社区，老牌社区
  • 菜鸟教程 - 常见的语言、技术教程合集
  • 掘金 - 有很多优质技术文章
  • OSCHINA - 中文开源技术交流社区
  • 博客园 - 常见的技社区，搬运内容较多

国外

  • StackOverFlow - 专为程序员提供的国际性问题解答交流社区
  • Reddit - 聚集了很多活跃的程序员大牛

其他工具

图片

  • removebg - 在线抠图工具，全自动且免费
  • 忙忙侠 - 免费AI抠图
  • 定稿 - 在线PS
  • TinyPNG - 在线压缩Png/Jpg文件
  • CloudConvert - 在线图片格式转换
  • 草料二维码 - 在线二维码生成、解析工具
  • carbon - 在线代码转图片工具，适用于代码显示不友好的情况下贴图使用
  • codeimg - 在线代码转图片工具
  • 百度识图 - 识别图片在互联网上的来源
  • 谷歌以图搜图 - 搜图工具，需要科学上网
  • ONLINE-CONVERT - 图片转ico
  • Apowersoft - 免费在线图片去水印
  • 稿定设计 - 免费在线公众号封面制作、朋友圈封面模板

视频

  • Capture - 开源本地录屏工具

实用API

  • FREE API - 免费API收集

电子书

  • 鸠摩搜索 - 搜索下载各种格式电子书

网络
